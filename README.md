Файл Main.kt, как инициализировать переменную, как использовать "шаблонные вставки" - мое устойчивое выражение,которое обозначает как вставлять значение переменной в строку 
---------------------------------------------------------------------------------------------
Конкретное задание: "Программа учета товаров на складе электроники"
Цель программы: Система должна отслеживать поступление, продажу и остатки товаров на складе, а также рассчитывать финансовые показатели.

Требования к функционалу:
1. Инициализация данных:
Объявить константы: максимальная вместимость склада, налоговая ставка
Создать переменные для учета: общее количество товаров, общая стоимость
Использовать отложенную инициализацию для некоторых переменных

2. Процесс приемки товара:
Добавить три вида товаров с разными характеристиками
Использовать augmented assignment для обновления количеств
Рассчитать общую стоимость запасов

3. Процесс продажи:
Реализовать продажу части товаров
Использовать augmented assignment для уменьшения остатков
Рассчитать выручку и прибыль

4. Отчетность:
Вывести детальную информацию с использованием строковых шаблонов
Показать финансовые расчеты
Отобразить статистику по складу


Как запустить - 
kotlinc Наименование_компилируемого_файла_с.расширением
kotlin Наименование_компилируемого_файла_сKt.Class
