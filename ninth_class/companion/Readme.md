Упрощенное задание: "Система учета студентов"
Задача (одна, но конкретная):
Создай класс Student для учета студентов в учебном заведении.

Требования:
Companion-объект в классе Student должен:
Хранить приватный счетчик totalStudents (сколько всего студентов создано)
Иметь публичный метод getTotalCount() для получения общего количества
Иметь приватный метод generateStudentId() который:
Увеличивает внутренний счетчик на 1
Возвращает ID в формате "STU-XXX", где XXX — номер студента (начиная с 001)
Формула: "STU-%03d".format(++studentCounter)

Класс Student должен иметь:
Свойство id: String (генерируется автоматически)
Свойство name: String (задается при создании)
Свойство year: Int (курс, от 1 до 5)
Конструктор, который принимает name: String и year: Int
Метод getInfo(): String, который возвращает: "Студент $name (ID: $id), $year курс"

В функции main():
Создай 3-4 студента
Выведи информацию о каждом
В конце выведи общее количество студентов
